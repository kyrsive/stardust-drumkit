# stardust-drumkit
a collection of rage rap styled drum sounds from [stardust](https://www.reddit.com/r/Drumkits/comments/rll8u9/playboi_carti_wlr_trippie_redd_rage_drumkit_over/) by Juicy Jules to be used on strudel

to use this inside of strudel, enter the following line in your strudel pattern:
```
samples('github:kyrsive/stardust-drumkit')
```

please [support Juicy Jules](https://www.instagram.com/prod.juicyjules/) if u can
